一、网站的类型

首先就是，我们要知道，网站主要区分为“静态网站”，“动态网站”

它们的区别主要在于内容是否会随着用户交互或后台逻辑而变化

1. 静态网站

页面内容是固定的，直接由 HTML、CSS、JavaScript 等文件组成。

2. 动态网站

页面内容由服务器端程序（如 PHP、Python Flask、Node.js 等）根据逻辑动态生成。

首先你还需要知道一个概念，后端类型

后端可以按照 运行方式 和 架构模式 来分类，不只是语言和框架。

传统后端：运行在固定服务器上（物理机或虚拟机），需要自己维护环境。
无服务器后端：不需要自己维护服务器，代码按需执行，自动扩展。
边缘计算后端：代码运行在全球分布的 CDN 节点，离用户更近。
微服务架构后端：系统拆分为多个小服务，每个服务独立运行，通常通过 API 通信。
后端即服务：直接使用第三方提供的后端功能，不需要自己写完整后端。
特殊类型后端：GraphQL 后端、Headless CMS、实时后端

购买服务器注意事项我也说一下吧

1.购买服务器的额外支出
公网带宽升级：默认带宽通常较低,也就是指访问速度，如果网站访问量大，需要额外购买更高带宽或按流量计费。（目前有两种，一种是按速度付费，一直是固定的速度，二种是按流量付费，花了多少访问的流量就给多少钱）
弹性 IP / 独立 IP：有些厂商默认提供共享IP，如果需要独立公网IP或多个IP，需要额外付费。
存储扩展：系统盘容量有限，网站数据多时需要购买额外数据盘或对象存储（OSS、COS）。
操作系统授权：Linux免费，Windows Server需额外授权费用。
！！！数据库服务：如果不自己搭建数据库，使用云数据库（RDS、MongoDB 等）需要单独付费。这是最最最重要的，很多人都忽略了数据库的费用，数据库的费用可能比服务器本身还贵，尤其是访问量大或者数据量大的时候。
！！！安全防护：DDoS防护、WAF（Web应用防火墙）、备份与快照、备案与合规。这里最重要的就是备案的问题，国内服务器必须备案。
！！！域名：域名注册和续费费用，通常按年计费，同时域名本身每年续费也是一笔固定支出。
