# 共创说明
所有人均可对在public目录下的所有文件进行创作和修改，欢迎大家积极参与，共同完善和丰富这些资源。

同时您需要知道的是下面的内容是严格审核的

| 目录              | 作用                                 |
|-----------------|------------------------------------|
| public/index.html  | 主页文件                               |
| public/index/*  | 存储index.html页面中所有需要用到的css,js,图片等文件 |
| public/school/* | 存储所有的班级公开页面文件和共创文件                 |

# 个人 贡献/页面 说明
所有自创页面需放在public/school/co-creation下，在该目录下创建一个自己的昵称文件夹，文件夹内随意创作

同时您需要知道的是，虽然您是可以随意创作，但是为了保证页面的质量和一致性，您的页面需要经过审核才能被正式收录。

同时完全禁止任何违法违规内容的创作，一经发现，立即删除。

具体的目录指引可在public/school/co-creation/co-creation.html中查看HSHMENG的创作提示

**注意！！！所有文件大小不得超过10MB，否则不予通过**
